
# Banana-Talks-Minion-Talks

It takes the input and convert into Minion language and display over page.


## Tech Stack

**Client:** HTML5, CSS3, VanilaJS, TailwindCSS



  
## Deployment

Deployed over netlify and can be accessed at 
https://miniontalksforclass.netlify.app/


  
## API Reference

#### Get converted string

```http
  GET /api/string
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |




Takes the input text and convert it into minion language.

  
## Features

- It lets user to interact
- Full responsive



 ## Description 
 - It asks for the text which you want to convert.
 - It coverts it into the minion language via api call.
 - It displays the converted thing over the page.
